# 1A2B Games

## 簡介 Introduction
### (此版本1A2B版數字可重複!)
剛開始學習Python時候寫的小遊戲，遊戲玩法就是猜數字，亂數產生4位數(可重複)，
**如果猜到的數字正確且位置正確則得到A，如果猜到的數字正確但位置錯誤則得到B，
通關條件就是4個A即獲勝**，獲勝之後可選擇重置繼續遊玩

----------------------------------------

## 環境設定 Environment
- 1.開發環境:**Python 3.7.8**
- 2.終端機執行 ```pip install -r requirements.txt``` 安裝會使用到的套件
- 3.使用工具:
    * 背景音樂撥放:```pygame==2.0.1```
    * GUI:```PyQt5==5.15.6```
